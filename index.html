Instalación y ejecución

Descargar la última versión del script de instalación:

  • Versión 0.3 ( Última versión ) ( Sistema Operativo soportado: Ubuntu 14.04 LTS )
  • Versión 0.2 ( Sistema Operativo soportado: Ubuntu 14.04 LTS )
  • Versión 0.1 ( Sistema Operativo soportado: Ubuntu 14.04 LTS )

Para la instalación:

sudo chmod 777 server_install_v0.3.sh

sudo ./server_install_v0.3.sh

Luego ejecutar el servidor con:

./server
